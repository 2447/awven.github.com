Database free blog

light blog is a free Markdown based blog. You write your posts in the simple markdown language and our clever php code does the rest!

Download »

Get it

We have two downloads available:

  • .zip

  • .tar.gz

  1. Now you need to install the whole folder onto your linux webserver.

  2. Make sure that there's a file called .htaccess. If not create one containing this code:

    
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond $1 !^(index\.php)
RewriteRule ^(.*)$ index.php/$1 [L]

  3. Now open app/config.ini in your favourite text editor. Change the site.url variable to the location of index.php. For example: http://example.com or http://example.com/folder/. Just remember to include a trailing slash

  4. Change the blog info to match your blog. Now save the file back onto your webserver

  5. Now open app/views/layout.html.php. You'll see some HTML code. Between the code <!-- the sidebar--> and <!--End custom sidebar--> you can add whatever you want. We've put a <ul> list in for you, but you can have anything!

  6. Now create you first post using Markdown. The syntax help is here. You must begin your document with #Title here otherwise it won't work!

  7. Call your file: YYYY-MM-DD_a-title.md

  8. Now refresh the page

  9. Your blog is complete!

Customize

Customize sidebar

Open app/views/layout.html.php. You'll see some HTML code. Between the code <!-- the sidebar--> and <!--End custom sidebar--> you can add whatever you want. We've put a <ul> list in for you, but you can have anything!

Customize theme

Open assets/css.style.css and make your changes there. If you write a really good theme let us know and we might add it to a theme chooser we are working on
